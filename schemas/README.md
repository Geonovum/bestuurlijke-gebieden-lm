# LGM-BG JSON Schema's

In deze map zijn de JSON Schema's te vinden die gebruikt worden voor het valideren van LGM-BG gegevens.

Er zijn twee varianten van de schema's:

* [LGM-BG.schema.json](LGM-BG.schema.json): Dit schema bevat het informatiemodel van LGM-BG inclusief directe relaties.
* [LGM-BG-attributes-only.schema.json](LGM-BG-attributes-only.schema.json): Dit schema bevat alleen de attributen van de objecttypen in LGM-BG, zonder relaties. Dit schema is handig voor situaties waarin alleen de attributen van objecttypen gevalideerd moeten worden en waar de relaties op andere manier worden vormgegeven, bijvoorbeeld via hypermedia links.
